# Lumerical-FDTD-Simulation-Based-Research
This repository is a curated, hands-on resource for learning Finite-Difference Time-Domain (FDTD) simulation in Ansys/Lumerical, built around a real nano-photonics case study rather than abstract textbook examples. It was created to support students working through FDTD fundamentals before applying them to their own research or coursework.
